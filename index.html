Loueurs de France

Loueurs de France est une entreprise familiale implantée à Avignon, spécialisée dans la location de véhicules utilitaires et de voitures. Forte de son expérience et de son engagement envers la satisfaction client, l'entreprise a récemment diversifié ses activités pour mieux répondre aux besoins de sa clientèle.

Elle propose désormais le dépôt-vente de véhicules, ce qui permet aux particuliers de vendre leur véhicule par son intermédiaire. De plus, Loueurs de France offre des services de vente et de montage de pneus, ainsi qu'un service dédié aux cartes grises, facilitant ainsi toutes les démarches administratives liées aux véhicules.

Mes responsabilités

En tant que responsable unique du projet, je gère l'ensemble du cycle de développement, de la conception à l'implémentation. Je conseille également l'entreprise sur sa stratégie digitale et son positionnement en ligne pour optimiser sa visibilité.

Mes Réalisations

Découvrez les projets concrets menés durant mon parcours en BTS SIO.

Première Année

Création d'un Site Web Initial

Développement d'un premier site vitrine pour l'entreprise. J'ai géré le prototypage, le design et l'intégration en utilisant HTML et CSS purs, dynamisé avec JavaScript vanilla. Le backend a été construit avec NodeJS et Express. Une attention particulière a été portée au SEO, à la gestion des erreurs et aux tests, avant déploiement sur un serveur de test.

HTML5 CSS3 JavaScript NodeJS Express SEO Tests Déploiement

Renforcement de la Sécurité

Mise en place de solutions pour améliorer la sécurité des données de l'entreprise. Création d'une base de données de mots de passe partagée et sécurisée avec Keepass, synchronisée via OneDrive, et renforcement de l'authentification avec la mise en place du 2FA (authentification à deux facteurs).

Sécurité Keepass OneDrive 2FA Gestion MDP

Optimisation Google Ads

Amélioration des campagnes publicitaires Google Ads existantes. J'ai affiné le ciblage en optimisant les mots-clés pour maximiser le volume pertinent, en excluant les termes non pertinents et en créant de nouvelles bannières publicitaires plus attractives pour augmenter le taux de clics.

Google Ads Marketing Digital SEO/SEA Optimisation

Deuxième Année

Refonte Complète du Site Web avec React

Modernisation du site avec une refonte totale utilisant React et Vite pour une expérience utilisateur dynamique et optimisée pour la conversion. Le backend Express a été renforcé (compression, sécurité CSP). Intégration d'une base de données MongoDB Atlas via Mongoose, avec routes admin protégées par JWT. Déploiement en production via Cpanel, incluant Google Analytics, tracking interne et gestion du consentement RGPD.

React Vite NodeJS Express MongoDB Mongoose JWT API Rest RGPD Analytics Cpanel

Conception d'un CMS sur Mesure

Développement d'un Système de Gestion de Contenu (CMS) en C# WPF. Cet outil desktop permet à l'entreprise de gérer de manière autonome les contenus clés du site (textes, véhicules, horaires, etc.) via une interface intuitive. L'authentification se fait via un token JWT communiqué aux routes sécurisées de l'API du site web.

C# WPF CMS Application Desktop API JWT Autonomie Client

Architecture Base de Données MongoDB

Mise en place et structuration d'une base de données NoSQL avec MongoDB. Conception de collections spécifiques pour chaque section majeure du site (véhicules, pages de contenu, etc.), assurant flexibilité et performance pour l'application web React et le CMS.

MongoDB NoSQL Base de Données Atlas Architecture
